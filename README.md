# Wraith::View

This will auto generate pages to view Wraith galleries, when they were created and enable a user to run a job.

## Installation

Add this line to your application's Gemfile:

    gem 'wraith-view'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install wraith-view
